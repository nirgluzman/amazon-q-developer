# MCP Server Usage Rules

## AWS Documentation MCP Server
Use `aws-documentation-mcp-server` when:
- Questions about AWS services, features, or APIs
- AWS best practices and architecture patterns
- AWS configuration and setup instructions
- AWS pricing, limits, or service quotas
- AWS CLI, SDK, or IaC (CloudFormation, CDK, Terraform) examples
- AWS security, compliance, or IAM policies

## Context7 MCP Server
Use `context7` when:
- Questions about third-party libraries and frameworks
- Programming language documentation (React, Node.js, Python, etc.)
- Open-source package usage and examples
- General programming concepts and patterns
- Non-AWS technical documentation
